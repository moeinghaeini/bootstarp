## 🛠️ **How to Use Auto-Deploy**

### **1. Setup (One-time):**
```bash
./setup-auto-deploy.sh
```

### **2. Start Auto-Deploy:**
```bash
npm run watch
```

### **3. Make Changes:**
- Edit any file in `src/`
- Auto-deploy triggers in 5 seconds
- Changes appear on AWS automatically

### **4. Manual Deploy:**
```bash
npm run deploy:aws
```
